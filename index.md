# esctheme

The `esctheme` package helps you create ggplot2 charts in the Essential
Services Commission of Victoria style.

### Installation

If you havenâ€™t already, install the `remotes` package:

`install.packages("remotes")`

Then install from GitHub:

``` r
# install.packages("devtools")
remotes::install_github("ESC-Economics/esctheme", dependencies = TRUE, upgrade = "always")
```

Then load into R:

[`library(esctheme)`](https://esc-economics.github.io/esctheme/)

### Using esctheme

The [package website](https://esc-economics.github.io/esctheme/)
contains a lot of information on how to use esctheme.

Start by reading the [Get
started](https://esc-economics.github.io/esctheme/articles/esctheme.md)
article.

There is also a wealth of information contained in the individual
[function
documentation](https://esc-economics.github.io/esctheme/reference/index.md)
that answers common usage questions.
