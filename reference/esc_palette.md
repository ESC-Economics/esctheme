# Predefined ESC colours combined into palettes

Predefined ESC colours combined into palettes

## Usage

``` r
esc_palette(n = 6, reverse = FALSE)
```

## Arguments

- n:

  Numeric. The number of levels in your colour scale, defaults to 6.
  Minimum value is 1, maximum is 12. Using more than 6 colours is not
  recommended as it may make it difficult to distinguish between
  colours.

- reverse:

  Logical. Reverse the standard colour order, defaults to FALSE
