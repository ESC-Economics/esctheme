# Save a ESC styled plot with sensible defaults

Save a ESC styled plot with sensible defaults

## Usage

``` r
esc_save(n, path = NULL, device = "png", width = 14.5, height = 6, ...)
```

## Arguments

- n:

  Name given to chart

- path:

  File path given to save chart

- device:

  "png" or "svg" file type

- width:

  The width size. Default to 14.5

- height:

  The height size. Default to 6

- ...:

  Optional arguments to ggsave.

## Value

Saved png file
